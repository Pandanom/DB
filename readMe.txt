Для налаштування програми для роботи з вашим сервером потрібно:
1)Відкрити файл з розширенням .config
2)У рядку conectionString після Data Source= написати назву вашого серверу
3)Після "User Id =" та "Password =" написати ваш логін та пароль до серверу відповідно
4)Приєднати базу даних Customers з архіву до вашого серверу
5)Почати роботу (Для входу в програму як адміністратор: виберіть в полі "Login" логін admin та у полі "password" введіть admin
далі можна буде змінити логін та пароль (дя цього натисніть на кнопку "Edit managers"))


//////////////////////////

Якщо ви вирішите змінити ім'я бази даних, то потрібно поміняти значення "Initial Catalog=Customers" 
на "Initial Catalog=Нове_ім'я"
